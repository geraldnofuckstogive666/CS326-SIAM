# TaskLite

Simple Task Management System

## Run Backend
```
cd backend
python -m pip install requirements.txt 
python -m pip install fastapi uvicorn
python -m uvicorn main:app --reload
```
## Open Frontend
```
Open frontend/index.html
open http://127.0.0.1:8000/ in your browser
```
